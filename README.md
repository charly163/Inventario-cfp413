# Sistema de Inventario CFP 413

Sistema de inventario para herramientas e insumos del CFP 413, con gestión de préstamos y configuración dinámica.

## Características

- ✅ Gestión completa de inventario
- ✅ Sistema de préstamos y devoluciones
- ✅ Registro de bajas y disposiciones
- ✅ Reportes y estadísticas
- ✅ Configuración personalizable
- ✅ Interfaz responsive

## Tecnologías

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Radix UI Components

## Instalación

1. Clona el repositorio
2. Instala las dependencias: `npm install`
3. Ejecuta el servidor de desarrollo: `npm run dev`
4. Abre [http://localhost:3000](http://localhost:3000)

## Versión

**Versión 36** - Sistema completo y funcional
